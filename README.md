# Node Restful API - MongoDB

## 1. Install MongoDB
	- Download & install
	- Add to Windows Path environment variable
	- Create directory - C:\data\db
	- Open terminal: mongod
	- Open another terminal: mongo
	
## 2. Open terminal (third):
	- npm install
	- gulp
	
## 3. Run - http://localhost:8000/api/
	- GET: /books
	- POST: /books
	- GET: /books/ID
	- PUT: /books/ID
	- PATCH: /books/ID
	- DELETE: /books/ID
	
	Format: 
	{
	"title": "Another Book",
	"author": "Someone",
	"genre": "Don`t know",
	"read": false
	}
